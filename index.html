Capstone Project

Locally — Project Overview

Project Summary

Locally is a local food service web application designed to connect farmers, restaurants, NGOs, and consumers through one platform. The project centers on sustainability, accessibility, and community trust.

Project Vision

The vision is to reduce food waste while supporting local economies, improving redistribution of surplus food, and giving users better visibility into food availability, usage, and donations.

Business / System Requirements

  • Marketplace for farmers to sell products directly
  • Restaurant surplus posting and reservation flows
  • Community features such as chats, groups, following, and notifications
  • Advanced search filters and review functionality
  • AI-driven usage tracking and recipe recommendations
  • Admin panel for moderation and verification
  • Secure payment integration
  • Responsive web app accessible on desktop and mobile browsers

Project Plan & Implementation Focus

1. Database schema design for products and reservations
2. Backend API development and authentication flows
3. Product and leftover posting features
4. NGO reservation flows and search/filter systems
5. UI/UX polish, testing, and deployment preparation

My documented responsibilities emphasize database design, system design, microservices support, DevOps, data analysis, and deployment contributions.
