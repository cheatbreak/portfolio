# Artem Basko Portfolio

A static portfolio website designed for deployment on Vercel.

## Files
- `index.html` — main website
- `styles.css` — styling
- `script.js` — mobile navigation toggle
- `assets/Artem_Basko_Resume.pdf` — resume download
- `assets/Artem_Basko_Cover_Letter_Template.txt` — sample cover letter template

## Deploy to Vercel
1. Create a new GitHub repository.
2. Upload all files from this folder to the repo.
3. Go to Vercel and click **Add New > Project**.
4. Import the GitHub repository.
5. Vercel should detect this as a static site.
6. Click **Deploy**.

## Customize
- Update `index.html` to add screenshots, more projects, transcript, recommendation letters, or awards.
- Replace resume and cover letter files in the `assets` folder when needed.
- Swap `Toronto, ON` with another public location if you want.

## Notes
This version keeps the public location general for privacy.
